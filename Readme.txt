 - 사용한 외부 모듈 : pillow, pprint, image, tkinter


 - 실행 방법

	1) 소스코드를 이용할 경우
	
		main.py를 실행합니다.
	
	2) exe파일을 이용할 경우
		
		/dist/main.exe를 실행합니다.
		(bookmark_character.json, item.json, logo.png, develop_logo.png파일이 없으면 실행되지 않습니다.)
	

 - 사용 방법

	1) 캐릭터 정보 검색
			
		서버와 캐릭터 이름을 입력한 뒤 검색버튼을 누릅니다.
		(서버와 캐릭터 둘 다 입력해야 진행됩니다.)

		서버, 캐릭터를 입력한 뒤 저장 버튼을 통해 즐겨찾기에 추가할 수 있습니다.


	2) 아이템 정보 검색

		검색창에 원하는 아이템의 이름을 입력하고 검색버튼을 누릅니다.
		아이템의 정보와 경매장 등록 정보를 열람할 수 있습니다.

		아이템 이름을 입력한 뒤 저장 버튼을 통해 즐겨찾기에 추가할 수 있습니다.
		

	3) 즐겨찾기
		
		즐겨찾기 리스트는 최대 5개까지 저장되며, 5개부터는 가장 오래된 데이터를 삭제하며 저장됩니다.
